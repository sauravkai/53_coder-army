# CipherSearch Design System

**Premium Light Theme** — Attractive, professional, applied to ALL pages.

---

## Pages Covered

| Page | Components Styled |
|------|-------------------|
| **Login** | Box, inputs, button, security note |
| **Dashboard** | Metrics, How It Works, crypto blocks, blockquote, use cases, activity |
| **Encrypt & Upload** | Buttons, inputs, client/server headers, doc list, expanders |
| **Search** | Query input, mode radio, pipeline, results, privacy metrics |
| **Security Proof** | Tabs, breach simulation, metrics, attack sim, expanders |
| **System Benchmark** | Metrics, benchmark cards, clear button |

---

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| **Background** | `#f8fafc` | Light, airy base |
| **Surface** | `#ffffff` | Cards, inputs |
| **Sidebar** | `#f1f5f9` | Nav area |
| **Primary** | `#0d9488` | Teal — CTAs, links, accents |
| **Secondary** | `#0891b2` | Cyan — gradient partner |
| **Tertiary** | `#7c3aed` | Violet — subtle highlights |
| **Success** | `#059669` | Green — secure, keys active |
| **Error** | `#dc2626` | Red — server zone, warnings |
| **Text** | `#0f172a` | Primary content |
| **Text Muted** | `#64748b` | Labels, captions |

**Gradient:** `#0d9488 → #0891b2 → #06b6d4` for buttons, progress, accents.

---

## Typography

- **Sans:** Plus Jakarta Sans
- **Mono:** JetBrains Mono (code, IDs)

---

## Key Styles

- **Cards:** White, soft shadow, 16px radius, teal border on hover
- **Buttons:** Gradient fill, lift on hover
- **Inputs:** White bg, focus ring
- **Sidebar:** Active item = teal tint + border
- **Blockquotes:** Left accent bar, light shadow
